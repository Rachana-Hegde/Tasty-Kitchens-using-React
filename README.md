# 🍽️ Tasty Kitchen – Food Ordering Website

## 📌 Overview

Tasty Kitchen is a responsive online food ordering web application that allows users to browse food items, view detailed information, add items to the cart, manage orders, and place food orders seamlessly. The website provides an intuitive user interface and a smooth user experience for customers looking to order food online.

## 🚀 Features

* User-friendly and responsive design
* Browse a variety of food categories
* View detailed food item information
* Add and remove items from the cart
* Update item quantities
* Calculate total order amount automatically
* Secure and smooth checkout process
* Mobile-friendly interface
* Fast navigation using React components

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router
* REST API Integration
* Bootstrap

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/tasty-kitchen.git
```

2. Navigate to the project folder

```bash
cd tasty-kitchen
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

5. Open your browser and visit

```text
http://localhost:3000
```
